第1章：心電図の基礎知識

心電図とは

心電図（ECG/EKG）は、心臓の電気的活動を記録する検査です。心筋細胞の脱分極と再分極の過程を波形として表示し、心臓の機能状態を評価できます。

  • • 非侵襲的検査：体表面から電極で記録
  • • リアルタイム評価：心臓の状態を即座に把握
  • • 標準化された記録：世界共通の読み方

心臓の電気伝導系

洞結節（SA node）

心臓のペースメーカー、60-100回/分

房室結節（AV node）

房室間の電気伝導を調節

His束・脚

心室への電気伝導路

Purkinje線維

心室筋への最終伝導

12誘導心電図の基本

肢誘導

  • • Ⅰ、Ⅱ、Ⅲ
  • • aVR、aVL、aVF
  • • 前額面の心電図

胸部誘導

  • • V1-V6
  • • 水平面の心電図
  • • 左室の詳細評価

記録条件

  • • 紙送り速度：25mm/秒
  • • 較正電圧：10mm/mV
  • • フィルター：0.05-100Hz

第2章：正常心電図の読み方

正常心電図の波形構成

P-QRS-T波群

  • • P波：心房脱分極（幅＜0.12秒、高さ＜2.5mm）
  • • QRS群：心室脱分極（幅0.06-0.10秒）
  • • T波：心室再分極
  • • PR間隔：0.12-0.20秒
  • • QT間隔：心拍数により変動

正常値の目安

心拍数

60-100回/分（安静時）

リズム

洞調律（規則正しいP-QRS-T）

軸偏位

正常軸：-30°〜+90°

ST-T変化

ST水平、T波陽性

第5章：徐脈性不整脈

洞性徐脈

洞性徐脈の特徴

  • • 心拍数：＜60回/分
  • • P波：正常（洞調律）
  • • RR間隔：規則的
  • • QRS：正常幅

徐脈の原因

  • • 生理的：睡眠、スポーツ心
  • • 薬剤性：β遮断薬、Ca拮抗薬
  • • 病的：洞不全症候群
  • • 代謝性：甲状腺機能低下症

第6章：伝導障害

房室ブロック

房室ブロックの分類

1度房室ブロック
  • • PR間隔：＞0.20秒
  • • 全てのP波に対してQRSが続く
  • • 通常無症状
2度房室ブロック
  • • Mobitz I型：PR間隔が徐々に延長
  • • Mobitz II型：PR間隔一定でQRS脱落
  • • 高度房室ブロック：2:1、3:1伝導
3度房室ブロック（完全房室ブロック）
  • • P波とQRSが完全に解離
  • • 房室結節性：40-60回/分
  • • 心室性：20-40回/分

第8章：特殊な症候群

WPW症候群

WPW症候群の特徴

  • • PR短縮：＜0.12秒
  • • δ波：QRS立ち上がりが緩徐
  • • QRS幅拡大：＞0.10秒
  • • 副伝導路：Kent束の存在

合併する不整脈

  • • 房室回帰性頻拍（AVRT）
  • • 心房細動（危険な場合あり）
  • • 心房粗動

第9章：電解質異常

カリウム異常

高カリウム血症

  • • T波増高：対称性、尖鋭化
  • • PR延長：P波平低化
  • • QRS拡大：重篤例
  • • 正弦波様：最重篤

低カリウム血症

  • • T波平低化：陰転化
  • • U波出現：T波後
  • • ST低下：軽度
  • • QT延長：見かけ上

第10章：実践問題

心電図読影の系統的アプローチ

基本チェック項目

  1. 1. リズム：規則性、洞調律
  2. 2. 心拍数：60-100回/分
  3. 3. P波：形状、PR間隔
  4. 4. QRS：幅、形状、軸
  5. 5. ST-T：上昇、低下、T波

重要な異常所見

  • • ST上昇：急性心筋梗塞
  • • ST低下：虚血、ジギタリス
  • • 異常Q波：陳旧性心筋梗塞
  • • 幅広QRS：心室性不整脈
  • • QT延長：薬剤、電解質異常

緊急度判定

緊急

  • • ST上昇型心筋梗塞
  • • 心室頻拍・細動
  • • 完全房室ブロック

準緊急

  • • NSTE-ACS
  • • 持続性心房細動
  • • 2度房室ブロック

経過観察

  • • 期外収縮
  • • 洞性不整脈
  • • 軽度ST-T変化

第3章：期外収縮（Premature Contractions）

期外収縮は最も頻度の高い不整脈で、正常な洞調律に加わる早期の異所性収縮です。 発生部位により上室性期外収縮（PAC）と心室性期外収縮（PVC）に分類されます。 健常者にも認められますが、病的な場合もあるため、適切な評価が重要です。

上室性期外収縮（PAC：Premature Atrial Contraction）

PACの特徴

  • • P波：早期に出現、形が正常と異なる
  • • QRS：正常幅（＜0.12秒）
  • • 代償性休止：不完全（短い）
  • • 頻度：健常者でも40-50%に認める

臨床的意義

  • 良性要因：ストレス、カフェイン、喫煙、睡眠不足
  • 病的要因：心房負荷、心房拡大、電解質異常
  • 予後：心房細動のリスク因子となることあり

診断のポイント

  • 1. P波の形：正常洞調律のP波と異なる
  • 2. 時期：予想されるよりも早い
  • 3. QRS：通常は正常形態
  • 4. 伝導：房室ブロックを伴うことあり

PACの種類と特徴

正常伝導型
  • • QRS正常形態
  • • 1:1房室伝導
  • • 最も頻度が高い
脚ブロック型
  • • QRS幅広い
  • • 機能的脚ブロック
  • • PVCと鑑別要
房室ブロック型
  • • P波のみでQRSなし
  • • 房室結節不応期
  • • blocked PACと呼ぶ

心室性期外収縮（PVC：Premature Ventricular Contraction）

PVCの特徴

  • • P波：先行するP波なし
  • • QRS：0.12秒以上の幅広い波形
  • • 代償性休止：完全（長い）
  • • T波：QRSと反対方向

PVCの重要な分類

形態による分類
  • 単形性：同一起源、比較的安全
  • 多形性：多起源、危険性が高い
  • 二方向性：交互に異なる形態
連発による分類
  • 単発：1個ずつ出現
  • 二段脈：1個おきに出現
  • 三段脈：2個おきに出現
  • 連発：2個以上連続（危険）
タイミングによる分類
  • R on T現象：T波上に出現、VFリスク
  • Late PVC：T波後に出現、比較的安全

Lownの分類（重症度評価）

Grade 0

PVCなし

Grade 1

散発性PVC

＜30個/時

Grade 2

頻発性PVC

≥30個/時

Grade 3

多形性PVC

形態が異なる

Grade 4-5

4a: couplet

4b: VT

5: R on T

臨床的意義
  • Grade 0-2：経過観察、症状があれば治療考慮
  • Grade 3-4a：心機能評価、治療検討
  • Grade 4b-5：緊急治療適応、VF移行リスク

PVCの起源同定

右室起源PVC
  • • V1でQS型またはrS型
  • • 下壁誘導で上向き
  • • 左脚ブロック様形態
  • • 流出路起源が多い
左室起源PVC
  • • V1でR型またはRS型
  • • 右脚ブロック様形態
  • • 心疾患に伴うことが多い
  • • より病的な意義

期外収縮の治療とフォローアップ

薬物療法

上室性期外収縮
  • • β遮断薬（第一選択）
  • • Ca拮抗薬（ベラパミル等）
  • • 抗不整脈薬（症状が強い場合）
心室性期外収縮
  • • β遮断薬（症状軽減）
  • • Ia群抗不整脈薬
  • • アミオダロン（重症例）
  • • ICD（高リスク例）

検査・評価

基本検査
  • • 12誘導心電図
  • • Holter心電図（24時間）
  • • 心エコー検査
  • • 血液検査（電解質）
追加検査（必要時）
  • • 運動負荷試験
  • • 心カテーテル検査
  • • 心筋シンチ
  • • 電気生理学的検査
