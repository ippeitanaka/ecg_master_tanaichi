第4章：頻脈性不整脈

心房細動（Atrial Fibrillation: AF）

心房細動の特徴

  • • P波：消失、f波（細動波）
  • • RR間隔：絶対性不整脈
  • • QRS：通常は正常幅
  • • 心拍数：100-180回/分

心房細動の分類

持続時間による分類
  • 発作性AF：7日以内に自然停止
  • 持続性AF：7日以上持続
  • 長期持続性AF：1年以上持続
  • 永続性AF：洞調律復帰を断念
心拍数による分類
  • 徐脈性AF：＜60回/分
  • 正常心拍数AF：60-100回/分
  • 頻脈性AF：＞100回/分

心房細動の重要な合併症

血栓塞栓症
  • • 脳梗塞（心原性脳塞栓症）
  • • 全身性塞栓症
  • • CHADS2、CHA2DS2-VAScスコア
心機能低下
  • • 頻脈誘発性心筋症
  • • 心房収縮の消失
  • • 心不全の悪化

心房粗動（Atrial Flutter: AFL）

心房粗動の特徴

  • • F波：のこぎり波、300回/分
  • • 房室伝導：通常2:1、時に可変
  • • 心拍数：典型的には150回/分
  • • RR間隔：比較的規則的

心房粗動の診断ポイント

  • 1. F波：Ⅱ、Ⅲ、aVFで明瞭なのこぎり波
  • 2. 心房レート：280-320回/分
  • 3. 房室伝導：2:1が最多、4:1も
  • 4. 迷走神経刺激：房室ブロック増悪でF波明瞭化

心室頻拍（Ventricular Tachycardia: VT）

心室頻拍の分類と特徴

持続時間による分類
  • 非持続性VT：30秒未満で自然停止
  • 持続性VT：30秒以上持続または血行動態悪化
形態による分類
  • 単形性VT：QRS形態が一定
  • 多形性VT：QRS形態が変化
  • Torsades de pointes：QTc延長に伴う
血行動態による分類
  • 血行動態安定：意識清明、血圧保持
  • 血行動態不安定：意識障害、ショック
基礎疾患
  • 器質的心疾患：冠動脈疾患、心筋症
  • 特発性VT：構造的心疾患なし

VTの心電図所見

  • • QRS幅：0.12秒以上（通常0.14秒以上）
  • • 心拍数：150-250回/分
  • • 房室解離：P波とQRSが独立
  • • capture beat：時に正常QRSが混入

第7章：虚血性心疾患

ST上昇型心筋梗塞（STEMI）

急性期の変化

  • • ST上昇：隣接2誘導以上で≥2mm
  • • 異常Q波：幅≥0.04秒、深さ>R波の1/3
  • • T波変化：高尖鋭化→陰転
  • • 経時的変化：数時間〜数日で進行

梗塞部位と誘導

前壁梗塞
  • 誘導：V1-V6
  • 責任血管：左前下行枝（LAD）
  • 特徴：最も頻度が高い
下壁梗塞
  • 誘導：Ⅱ、Ⅲ、aVF
  • 責任血管：右冠動脈（RCA）
  • 特徴：徐脈、房室ブロック合併
側壁梗塞
  • 誘導：Ⅰ、aVL、V5、V6
  • 責任血管：左回旋枝（LCX）
  • 特徴：心不全合併しやすい

心筋梗塞の経時的変化

超急性期（数分）
  • • 高尖鋭T波
  • • 軽度ST上昇
急性期（数時間）
  • • 明瞭なST上昇
  • • 異常Q波出現
亜急性期（数日）
  • • ST正常化
  • • T波陰転
慢性期（数週間）
  • • 異常Q波残存
  • • T波正常化

非ST上昇型急性冠症候群（NSTE-ACS）

NSTE-ACSの特徴

  • • ST低下：≥1mm、水平または下降傾斜
  • • T波陰転：深い対称性陰転
  • • 異常Q波：通常は認めない
  • • 一過性変化：症状軽快で正常化

NSTE-ACSの分類

不安定狭心症（UA）
  • • 心筋逸脱酵素上昇なし
  • • トロポニン陰性
  • • 可逆性心筋虚血
非ST上昇型心筋梗塞（NSTEMI）
  • • 心筋逸脱酵素上昇
  • • トロポニン陽性
  • • 心筋壊死あり
